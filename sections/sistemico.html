Pensamento Sistêmico

Pensamento Sistêmico is my current project. It stands for "Systems Thinking" in Portuguese. It's a digital garden on topics related to Cybernetics.The project has been built on top of the Zettelgarden, and each note is written directly in Markdown via Logseq.

Since I described this a lot of its features and motivations in the previous application field, so I'll save you from redundancy.
